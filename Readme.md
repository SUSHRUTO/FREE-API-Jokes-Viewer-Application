# 😂 JokeVerse – Professional Jokes Viewer Application

JokeVerse is a modern and interactive Jokes Viewer Application built using HTML, Tailwind CSS, and Vanilla JavaScript.

The application fetches jokes dynamically using the FreeAPI Random Jokes endpoint and presents them in a clean, engaging, and professional user interface.

It also includes custom joke generation, category filtering, search functionality, random joke generation, and responsive design.
